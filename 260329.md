# 用户登录后界面改造记录

## 日期
2026-03-29

## 需求描述
修改用户登录后界面，实现以下功能：
1. 左侧栏从上到下依次显示：新建笔记、新建文件夹、搜索框、分隔线、文件浏览器（类似VSCode/Obsidian的树形文件管理）
2. 登录后主界面中间显示全部笔记
3. 点击左侧栏文件夹后，左侧栏展开文件夹，主界面中间显示该文件夹下的笔记
4. 点击左侧栏笔记或中间笔记列表中的笔记时，中间部分替换为笔记编辑界面

## 实现方案

### 1. 创建文件树组件 FileTree.vue

**文件路径**: `calcite-web/src/components/FileTree.vue`

**核心功能**:
- 递归渲染文件夹树形结构
- 支持文件夹展开/折叠
- 显示文件夹内的笔记
- 右键菜单支持：新建笔记、新建子文件夹、重命名、删除
- 选中状态高亮

**关键API调用**:
- `getFolderList({ folder_id: parentId })` - 获取子文件夹
- 笔记通过props传入，按folder_id筛选

### 2. 重构 Home.vue

**文件路径**: `calcite-web/src/views/Home.vue`

**主要改动**:

#### 左侧栏布局
```
┌─────────────────────────┐
│  [新建笔记] 按钮         │
│  [新建文件夹] 按钮        │

│  ─────────────────       │ 分隔线
│  ┌─ 文件夹A            │
│  │   ├─ 子文件夹B      │
│  │   └─ 笔记1          │
│  └─ 笔记2              │
│  ─────────────────       │

└─────────────────────────┘
```

#### 中间区域状态切换
- **笔记列表视图**: 显示全部笔记或选中文件夹的笔记
- **笔记编辑视图**: 替换列表视图，显示笔记编辑器

#### 核心功能实现

**数据获取**:
```javascript
// 递归获取所有文件夹
const fetchFoldersRecursive = async (parentId, currentFolders = []) => {
  const result = [...currentFolders]
  for (const folder of currentFolders) {
    const childData = await getFolderList({ folder_id: folder.id })
    if (childData?.length > 0) {
      const children = await fetchFoldersRecursive(folder.id, childData)
      result.push(...children)
    }
  }
  return result
}
```

**展开状态管理**:
```javascript
const expandedFolders = ref(new Set())

const handleFolderExpand = (folder) => {
  expandedFolders.value.add(folder.id)
}
```

**编辑器集成**:
- 使用 `editingNote` 控制编辑器显示
- 自动保存机制：输入后2秒自动保存
- 未保存更改提示

**自动展开父文件夹**:
```javascript
const expandParentFolders = (parentId) => {
  if (!parentId || parentId === 0) return
  expandedFolders.value.add(parentId)
  const parent = allFolders.value.find(f => f.id === parentId)
  if (parent?.parent_id) {
    expandParentFolders(parent.parent_id)
  }
}
```

## API对接

### 已使用的API

| 接口 | 方法 | 用途 |
|------|------|------|
| /api/folder/list | GET | 获取文件夹列表（支持parent_id） |
| /api/note/list | GET | 获取笔记列表（支持folder_id） |
| /api/note/detail | GET | 获取笔记详情 |
| /api/note/create | POST | 创建笔记 |
| /api/note/update | POST | 更新笔记 |
| /api/note/delete | POST | 删除笔记 |
| /api/note/search | GET | 搜索笔记 |
| /api/folder/create | POST | 创建文件夹 |
| /api/folder/update | POST | 更新文件夹 |
| /api/folder/delete | POST | 删除文件夹 |
