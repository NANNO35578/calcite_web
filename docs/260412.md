# 2026-04-12 笔记编辑器 Markdown 改造

## 概述
将笔记编辑视图中的文本编辑器替换为 Markdown 编辑器，使用 `md-editor-v3` 组件，并适配 Everforest 主题。

## 修改文件

### 1. `calcite-web/src/components/center/NoteEditor.vue`

#### 变更内容：
- **替换输入组件**: 将原生的 `el-input` 文本域替换为 `md-editor-v3` Markdown 编辑器
- **新增导入**: 
  ```javascript
  import MdEditor from 'md-editor-v3'
  import 'md-editor-v3/lib/style.css'
  ```
- **工具栏配置**: 配置了常用的 Markdown 编辑工具栏
  ```javascript
  const toolbars = [
    'bold', 'underline', 'italic', 'strikeThrough',
    'sub', 'sup', 'quote', 'unorderedList', 'orderedList',
    'codeRow', 'code', 'link', 'image', 'table',
    'preview', 'fullscreen'
  ]
  ```

#### Everforest 主题适配：
通过深度选择器 (`:deep`) 覆盖了 `md-editor-v3` 的默认样式，适配 Everforest 主题配色：

| 组件部分 | 背景色 | 文字色 |
|---------|--------|--------|
| 编辑器容器 | `--bg-secondary` (#2d353b) | - |
| 工具栏 | `--bg-tertiary` (#414b50) | `--text-secondary` |
| 编辑区域 | `--bg-secondary` | `--text-primary` (#d3c6aa) |
| 预览区域 | `--bg-secondary` | `--text-primary` |
| 代码块 | `--bg-tertiary` | `--accent-primary` (#7fbbb3) |
| 引用块 | `--bg-tertiary` | `--text-secondary`，左边框 `--accent-secondary` |
| 表格头部 | `--bg-hover` | `--accent-primary` |

#### 样式细节：
- 工具栏悬停效果：背景 `--bg-hover`，文字 `--accent-primary`
- 链接颜色：`--accent-primary`，悬停 `--accent-secondary`
- 列表标记：`--accent-secondary`
- 滚动条：与系统滚动条样式保持一致

## 依赖说明
项目已安装 `md-editor-v3` (v6.4.2)，这是一个专为 Vue 3 设计的 Markdown 编辑器组件，支持：
- 实时预览
- 全屏编辑
- 丰富的工具栏
- 代码高亮
- 主题自定义

## 界面效果
- 标题输入框保持原有样式
- 内容区域变为 Markdown 双栏编辑器（编辑 + 预览）
- 整体配色与 Everforest 深色主题和谐统一
- 支持全屏编辑模式
