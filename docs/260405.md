## 实现内容

### 1. 搜索API (`calcite-web/src/api/note.js`)
- 更新注释，明确支持 `from`（分页起始）和 `size`（每页条数）参数

### 2. 搜索功能增强 (`calcite-web/src/views/Home.vue`)

**新增状态变量：**
```javascript
searchTotal      // 搜索结果总数
searchFrom       // 分页起始位置
searchPageSize   // 每页条数（默认20）
```

**新增方法：**
- `doSearch(resetPage)` - 核心搜索函数，带分页参数
- `handleSearchPrev()` - 上一页
- `handleSearchNext()` - 下一页

**UI 功能：**
- ✅ 搜索框输入 + 防抖处理（300ms）
- ✅ 加载状态展示（骨架屏）
- ✅ 无数据提示（带友好提示语）
- ✅ 高亮标题渲染 (`highlight_title`)
- ✅ 高亮内容渲染 (`highlight_content`)
- ✅ 更新时间格式化显示
- ✅ 相关度分数显示 (`score`)
- ✅ 分页控件（上一页/下一页 + 页码显示）

**新增图标：**
- `ArrowRight`, `Clock`, `Star`

### 3. 样式完善
- 搜索结果头部样式（标题 + 总数 + 分页控件）
- 高亮标签样式 (`<mark>` 标签高亮显示)
- 元信息样式（时间、分数）
- 无数据状态样式

### 4. API 封装
`request.js` 已统一处理：
- 请求头自动附加 `Authorization: Bearer {token}`
- 响应code非0时自动提示错误
- 401状态码自动跳转登录页
- 网络错误友好提示
