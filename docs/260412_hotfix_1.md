# 2026-04-12 Hotfix 1 - 修复 md-editor-v3 导入错误

## 问题描述
浏览器打开为白色页面，Console 报错：
```
Uncaught SyntaxError: The requested module '/node_modules/.vite/deps/md-editor-v3.js?v=1d3cc0bc' 
does not provide an export named 'default' (at NoteEditor.vue:49:8)
```

## 问题原因
`md-editor-v3` 库使用的是**命名导出 (Named Export)** `MdEditor`，而不是默认导出 (Default Export)。

在库的源码 `node_modules/md-editor-v3/lib/es/index.mjs` 中：
```javascript
export {
  ze as DropdownToolbar,
  nr as MdCatalog,
  Ve as MdEditor,  // <-- 命名导出
  Qe as MdModal,
  ar as MdPreview,
  // ...
};
```

## 修复操作

### 修改文件：`calcite-web/src/components/center/NoteEditor.vue`

**错误代码（第 49 行）：**
```javascript
import MdEditor from 'md-editor-v3'
```

**修复后代码：**
```javascript
import { MdEditor } from 'md-editor-v3'
```

## 经验教训
1. 在导入第三方库时，应先确认其导出方式（默认导出 vs 命名导出）
2. 可以通过查看 `node_modules/<package>/lib/es/index.mjs` 确认导出方式
3. VS Code 的自动导入功能可能会错误地选择默认导入，需要手动检查

## 验证方法
修复后刷新浏览器，页面应正常显示，Markdown 编辑器可正常加载。
