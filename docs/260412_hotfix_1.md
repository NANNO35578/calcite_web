# Calcite Web 布局留白修复文档

**日期**: 2026-04-12  
**版本**: Hotfix v1.0  
**问题**: ElSplitter 三栏布局无法铺满屏幕、出现留白

---

## 问题现象

使用 `el-splitter` 三栏布局后：
1. **右侧栏区域**没有铺满父容器，出现空白
2. **左侧栏下方**出现留白
3. 三栏无法完全填满整个视口

---

## 根因分析

### 1. 侧边栏组件高度未填满

**LeftSidebar.vue** 和 **RightSidebar.vue** 的根元素 `.sidebar-content` 缺少 `height: 100%`，导致无法填满父容器：

```css
/* 修复前 */
.sidebar-content {
  flex: 1;
  display: flex;
  flex-direction: column;
}

/* 修复后 */
.sidebar-content {
  width: 100%;
  height: 100%;  /* 关键修复 */
  display: flex;
  flex-direction: column;
  overflow: hidden;
}
```

### 2. ElSplitter Pane 配置问题

**修复前**的配置使用了数组绑定和百分比字符串混用：
```vue
<el-splitter v-model="splitterSizes">
  <el-splitter-pane :size="splitterSizes[0]" min-size="25%" max-size="50%">
  <el-splitter-pane :size="splitterSizes[1]" min-size="50%" max-size="70%">
  <el-splitter-pane :size="splitterSizes[2]" min-size="25%" max-size="50%">
```

**修复后**使用统一的百分比格式：
```vue
<el-splitter>
  <el-splitter-pane size="25%" :min="200" :max="400">
  <el-splitter-pane size="50%" :min="300">
  <el-splitter-pane size="25%" :min="200" :max="400">
```

### 3. 根容器尺寸设置

**修复前**的 `.home-container` 缺少明确的视口尺寸：
```css
.home-container {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  display: flex;
}
```

**修复后**添加明确的 `width: 100vw` 和 `height: 100vh`：
```css
.home-container {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  width: 100vw;      /* 明确视口宽度 */
  height: 100vh;     /* 明确视口高度 */
  display: flex;
  overflow: hidden;  /* 防止滚动条 */
}
```

### 4. TagList 最大高度限制

**TagList.vue** 的 `max-height: 300px` 限制了标签列表的扩展：
```css
/* 修复前 */
.tag-list {
  max-height: 300px;  /* 限制高度导致留白 */
  overflow-y: auto;
}

/* 修复后 */
.tag-list {
  flex: 1;            /* 自动填满剩余空间 */
  overflow-y: auto;
  min-height: 0;      /* 允许 flex 子项收缩 */
}
```

## 修复详情

### 文件 1: Home.vue

#### 修改 1: ElSplitter 配置
```vue
<!-- 修复前 -->
<el-splitter v-model="splitterSizes" class="main-splitter">
  <el-splitter-pane :size="splitterSizes[0]" min-size="25%" max-size="50%">
  <el-splitter-pane :size="splitterSizes[1]" min-size="50%" max-size="70%">
  <el-splitter-pane :size="splitterSizes[2]" min-size="25%" max-size="50%">

<!-- 修复后 -->
<el-splitter class="main-splitter">
  <el-splitter-pane size="25%" :min="200" :max="400">
  <el-splitter-pane size="50%" :min="300">
  <el-splitter-pane size="25%" :min="200" :max="400">
```

#### 修改 2: 删除未使用的变量
```javascript
// 删除
const splitterSizes = ref([25, 50, 25])
```

## 布局参数

### 三栏比例 (1:2:1)

| 位置 | 默认比例 | 默认宽度 | 最小宽度 | 最大宽度 |
|------|---------|---------|---------|---------|
| 左侧栏 | 25% | 25% | 200px | 400px |
| 中间栏 | 50% | 50% | 300px | - |
| 右侧栏 | 25% | 25% | 200px | 400px |

### 铺满策略

1. **根容器** (`home-container`): `100vw × 100vh`
2. **ElSplitter**: `width: 100%, height: 100%, flex: 1`
3. **Pane 子元素**: 每个都设置 `height: 100%`
4. **侧边栏内容**: `width: 100%, height: 100%`
5. **中间内容区**: `width: 100%, height: 100%`

## 关键样式总结

### 铺满三要素

1. **视口根元素**: `width: 100vw; height: 100vh`
2. **Flex 父元素**: `display: flex; flex-direction: column`
3. **Flex 子元素**: `flex: 1; min-height: 0` (水平方向同理)

### ElSplitter 正确使用

```vue
<el-splitter class="main-splitter">
  <el-splitter-pane size="25%" :min="200" :max="400">
    <!-- 内容必须有 height: 100% -->
    <div style="height: 100%">...</div>
  </el-splitter-pane>
</el-splitter>
```

```css
.main-splitter {
  width: 100%;
  height: 100%;
}

.main-splitter :deep(.el-splitter__pane) {
  height: 100%;
}
```
