# Calcite Web 修复文档 - SplitterPanel 没有 toggle 方法

**日期**: 2026-04-12  
**版本**: Hotfix v7.0  
**问题**: rightPanel.value?.toggle is not a function

---

## 问题根因

Element Plus 的 `el-splitter-panel` **没有 `toggle` 方法**，它只暴露了 `splitterPanelRef` 属性。

错误信息：
```
TypeError: rightPanel.value?.toggle is not a function
```

---

## 解决方案

不使用 ref 调用方法，而是直接通过 `size` 属性控制：

```vue
<el-splitter-panel :size="leftCollapsed ? '0px' : '25%'">
```

---

## 修复详情

### 修改前

```vue
<el-splitter-panel ref="leftPanel" size="25%" collapsible>

<script>
const leftPanel = ref()
const toggleLeft = () => leftPanel.value?.toggle()
</script>
```

### 修改后

```vue
<el-splitter-panel :size="leftCollapsed ? '0px' : '25%'" :min="0">

<script>
// 直接通过 size 属性控制，无需 ref
</script>
```

---

## 关键改动

| 文件 | 修改 |
|------|------|
| `Home.vue` | 删除 `ref` 和 `toggle` 方法 |
| `Home.vue` | 改用 `:size="collapsed ? '0px' : '25%'"` |
| `Home.vue` | 设置 `:min="0"` 允许完全折叠 |

---

## 折叠原理

- 展开时：`size="25%"`
- 折叠时：`size="0px"`
- `:min="0"` 允许面板收缩到 0
