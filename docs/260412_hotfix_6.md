# Calcite Web 修复文档 - Panel Toggle 方法

**日期**: 2026-04-12  
**版本**: Hotfix v6.0  
**问题**: el-splitter-panel 不支持 v-model 控制折叠

---

## 问题根因

`el-splitter-panel` 组件**不支持**通过 `:collapsed` 属性或 `v-model` 控制折叠状态。

只能通过 **ref 实例的 `.toggle()` 方法**触发折叠/展开。

---

## 修复方案

### 1. 给面板添加 ref

```vue
<el-splitter-panel ref="leftPanel" size="25%" :min="200" collapsible>
<el-splitter-panel ref="rightPanel" size="25%" :min="200" collapsible>
```

**注意**: 删除 `:collapsed` 和 `@collapse/@expand` 绑定

---

### 2. 添加 ref 和 toggle 方法

```javascript
// ===== Panel 实例引用 =====
const leftPanel = ref()
const rightPanel = ref()

// ===== 切换面板折叠/展开 =====
const toggleLeft = () => leftPanel.value?.toggle()
const toggleRight = () => rightPanel.value?.toggle()
```

---

### 3. 绑定事件

```vue
<CenterToolbar
  @toggle-left="toggleLeft"
  @toggle-right="toggleRight"
/>
```
