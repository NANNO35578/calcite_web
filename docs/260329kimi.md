## 第二部分：主题系统重构

### 设计目标

将分散在 `Home.vue` 中的主题配色（约 400 行代码）提取为独立的主题系统，方便：
1. 代码逻辑编写 - 使用语义化变量名
2. 配色设置 - 集中管理深浅色主题
3. 扩展新主题 - 只需添加配置对象

### 创建的文件

#### 1. `src/styles/theme.js` - 主题配置中心

定义了完整的主题结构：

```javascript
// 主题配置对象结构
{
  name: 'dark',
  label: '深色',
  bg: { primary, secondary, tertiary, hover, active },      // 背景色
  text: { primary, secondary, muted, inverse },              // 文字色
  accent: { primary, secondary, warning, error },            // 强调色
  border: { primary, secondary, accent },                    // 边框色
  component: { folder, note, tag, scrollThumb }              // 组件色
}
```

提供的工具函数：
- `initTheme()` - 初始化主题
- `toggleTheme()` - 切换主题
- `setTheme(name)` - 设置指定主题
- `applyTheme(name)` - 应用主题到 DOM

#### 2. `src/styles/theme.css` - CSS 变量定义

包含：
- CSS 变量默认值（深色主题）
- Element Plus 组件样式覆盖
- 滚动条样式

使用方式：
```css
.my-class {
  background-color: var(--bg-primary);
  color: var(--text-primary);
  border: 1px solid var(--border-primary);
}
```

#### 3. `src/composables/useTheme.js` - 组合式函数

在 Vue 组件中使用：

```javascript
import { useTheme } from '../composables/useTheme'

const { 
  currentTheme,    // 当前主题名
  isDark,          // 是否深色
  isLight,         // 是否浅色
  themeConfig,     // 当前主题配置
  availableThemes, // 所有可用主题
  toggleTheme,     // 切换主题
  setTheme         // 设置主题
} = useTheme()
```

### 修改的文件

#### 1. `src/main.js`

添加主题初始化和样式引入：

```javascript
import './styles/theme.css'
import { initTheme } from './styles/theme'

initTheme()
```

#### 2. `src/views/Home.vue`

**简化前**：约 1850 行，其中样式约 780 行，包含大量重复的媒体查询

**简化后**：约 1080 行，样式约 350 行

主要修改：
- 所有硬编码颜色替换为 CSS 变量
- 删除约 400 行的 `@media (prefers-color-scheme: light)` 媒体查询
- 保留结构和布局样式

**颜色映射示例**：

| 原硬编码 | CSS 变量 |
|---------|---------|
| `#272e33` | `var(--bg-primary)` |
| `#2d353b` | `var(--bg-secondary)` |
| `#414b50` | `var(--bg-tertiary)` |
| `#d3c6aa` | `var(--text-primary)` |
| `#9da9a0` | `var(--text-secondary)` |
| `#7a8478` | `var(--text-muted)` |
| `#7fbbb3` | `var(--accent-primary)` |
| `#a7c080` | `var(--accent-secondary)` |

#### 3. `src/components/FileTree.vue`

将硬编码颜色和 CSS 变量后备值替换为统一的 CSS 变量：

```css
/* 修改前 */
.folder-icon {
  color: var(--folder-icon-color, #a7c080);
}

/* 修改后 */
.folder-icon {
  color: var(--component-folder);
}
```

### 主题系统优势

1. **代码量减少**：Home.vue 样式从 ~780 行减少到 ~350 行
2. **易于维护**：颜色集中管理，修改一处全局生效
3. **动态切换**：支持运行时切换主题，无需刷新页面
4. **扩展性好**：添加新主题只需在 `theme.js` 中添加配置对象
5. **语义化命名**：使用 `bg-primary`、`text-secondary` 等语义化名称，代码更易读

2. **跟随系统主题**：监听 `prefers-color-scheme` 自动切换
3. **自定义主题**：允许用户自定义颜色并保存
