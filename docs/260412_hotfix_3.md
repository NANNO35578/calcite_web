# Calcite Web 修复文档 - Vue 指令警告

**日期**: 2026-04-12  
**版本**: Hotfix v3.0  
**问题**: Runtime directive used on component with non-element root node

---

## 问题描述

浏览器 Console 警告：
```
[Vue warn]: Runtime directive used on component with non-element root node. 
The directives will not function as intended.
  at <ElSplitterPanel size="25%" min=200 max=400>
```

**原因**: 在 `el-splitter-panel` 组件上使用了 `v-show` 指令，但该组件的根节点不是原生 DOM 元素，导致指令无法正常生效。

---

## 修复方案

将 `v-show` 从 `el-splitter-panel` 组件移到其内部的包裹元素上。

### 修复前

```vue
<el-splitter-panel v-show="!leftCollapsed" size="25%">
  <LeftSidebar ... />
</el-splitter-panel>

<el-splitter-panel v-show="!rightCollapsed" size="25%">
  <RightSidebar ... />
</el-splitter-panel>
```

### 修复后

```vue
<el-splitter-panel size="25%">
  <div v-show="!leftCollapsed" class="panel-content">
    <LeftSidebar ... />
  </div>
</el-splitter-panel>

<el-splitter-panel size="25%">
  <div v-show="!rightCollapsed" class="panel-content">
    <RightSidebar ... />
  </div>
</el-splitter-panel>
```

---

## 新增样式

```css
.panel-content {
  width: 100%;
  height: 100%;
  overflow: hidden;
}
```
