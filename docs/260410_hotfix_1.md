## 修复内容

### 1. 修复列表挤压问题 (Fix Layout Compression)

**问题描述**: 当笔记/文件数量过多时，中间笔记列表、左侧目录树条目被挤压成极小长条。

**解决方案**: 
- 给列表容器设置固定高度 (`flex: 1` + `overflow-y: auto`)
- 超出内容内部滚动，条目保持最小高度不挤压
- **只让容器滚动，不让单条卡片内部出现滚动条**

**关键代码**:

```css
/* 文件树容器 - 固定高度，内部滚动 */
.file-tree-container {
  flex: 1;
  overflow-y: auto;
  padding: 8px;
  min-height: 0; /* 关键：防止 flex 子项溢出 */
}

/* 笔记列表容器 - 固定高度，内部滚动 */
.note-list {
  flex: 1;
  overflow-y: auto;
  display: flex;
  flex-direction: column;
  gap: 12px;
  min-height: 0; /* 关键：防止 flex 子项溢出 */
}

/* 笔记卡片 - 保持最小高度，不允许内部滚动 */
.note-card {
  background-color: var(--bg-secondary);
  border: 1px solid var(--border-primary);
  cursor: pointer;
  transition: all 0.2s;
  flex-shrink: 0; /* 关键：防止被压缩 */
}

/* 标签列表容器 - 固定高度 */
.tag-list {
  max-height: 300px; /* 限制最大高度 */
  overflow-y: auto;
  padding: 12px 16px;
  margin-bottom: 16px;
}
```

### 2. 新增左右栏折叠功能 (Sidebar Collapse)

**功能描述**: 仿 VS Code 风格的侧边栏折叠功能

**特性**:
- 按钮放在栏内侧垂直居中
- 点击可平滑收起/展开 (CSS transition)
- 中间区域自动填充宽度 (flex: 1)
- 折叠状态用 localStorage 保存 (持久化)

**关键代码**:

```javascript
// ===== 侧边栏折叠状态（从 localStorage 读取，默认展开）=====
const leftCollapsed = ref(localStorage.getItem('calcite:leftCollapsed') === 'true')
const rightCollapsed = ref(localStorage.getItem('calcite:rightCollapsed') === 'true')

// 监听折叠状态变化，保存到 localStorage
watch(leftCollapsed, (val) => {
  localStorage.setItem('calcite:leftCollapsed', String(val))
})
watch(rightCollapsed, (val) => {
  localStorage.setItem('calcite:rightCollapsed', String(val))
})
```

```css
/* 左侧栏和右侧栏基础样式 */
.sidebar {
  display: flex;
  flex-direction: column;
  position: relative;
  background-color: var(--bg-secondary);
```

## 技术要点

### 1. Flexbox 防止挤压的关键

```css
.container {
  display: flex;
  flex-direction: column;
  min-height: 0; /* 必须设置，否则子项会溢出 */
}

.scrollable-area {
  flex: 1;
  overflow-y: auto;
  min-height: 0; /* 必须设置，防止内容撑开 */
}

.item {
  flex-shrink: 0; /* 防止被压缩 */
}
```

### 2. CSS 过渡动画

```css
.sidebar {
  transition: width 0.3s cubic-bezier(0.25, 0.8, 0.25, 1);
}
```

使用 `cubic-bezier` 缓动函数实现更自然的展开/收起动画。

### 3. localStorage 持久化

```javascript
// 读取
const value = localStorage.getItem('key') === 'true'

// 写入
localStorage.setItem('key', String(value))
```

注意：localStorage 只能存储字符串，布尔值需要转换。
