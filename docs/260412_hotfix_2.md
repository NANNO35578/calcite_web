# Calcite Web 布局修复文档 - 第二轮

**日期**: 2026-04-12  
**版本**: Hotfix v2.0  
**问题**: 
1. 浏览器 console 警告: `Failed to resolve component: el-splitter-pane`
2. 依旧没有铺满屏幕

---

## 问题 1: 组件未找到警告

### 错误信息
```
[Vue warn]: Failed to resolve component: el-splitter-pane
If this is a native custom element, make sure to exclude it from component resolution via compilerOptions.isCustomElement.
```

### 根因
Element Plus 的 Splitter 子组件名称是 `el-splitter-panel`，而不是 `el-splitter-pane`。

### 修复
**文件**: `Home.vue`
```vue
<!-- 修复前 -->
<el-splitter-pane size="25%">

<!-- 修复后 -->
<el-splitter-panel size="25%">
```

**文件**: `main.js`
添加显式组件注册：
```javascript
import { ElSplitter, ElSplitterPanel } from 'element-plus'

app.component('ElSplitter', ElSplitter)
app.component('ElSplitterPanel', ElSplitterPanel)
```

---

## 问题 2: 样式类名错误

### 根因
CSS 中使用了错误的类名 `.el-splitter__pane`，应该是 `.el-splitter__panel`。

### 修复
**文件**: `Home.vue`
```css
/* 修复前 */
.main-splitter :deep(.el-splitter__pane) {
  overflow: hidden;
  height: 100%;
}

/* 修复后 */
.main-splitter :deep(.el-splitter__panel) {
  overflow: hidden;
  height: 100%;
}
```

## 完整修复清单

### 修改文件 1: main.js
```javascript
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import { ElSplitter, ElSplitterPanel } from 'element-plus'  // 新增
import 'element-plus/dist/index.css'
import router from './router'
import App from './App.vue'
import './style.css'
import './styles/theme.css'
import { initTheme } from './styles/theme'

initTheme()

const app = createApp(App)

app.use(ElementPlus)
app.component('ElSplitter', ElSplitter)         // 新增
app.component('ElSplitterPanel', ElSplitterPanel) // 新增
app.use(router)
app.mount('#app')
```

### 修改文件 2: Home.vue

#### Template 部分
```vue
<el-splitter class="main-splitter">
  <!-- 使用 el-splitter-panel 而非 el-splitter-pane -->
  <el-splitter-panel size="25%" :min="200" :max="400">
    <LeftSidebar ... />
  </el-splitter-panel>
  
  <el-splitter-panel size="50%" :min="300">
    <div class="main-content">...</div>
  </el-splitter-panel>
  
  <el-splitter-panel size="25%" :min="200" :max="400">
    <RightSidebar ... />
  </el-splitter-panel>
</el-splitter>
```

#### Style 部分
```css
/* ElSplitter Panel：确保内容区填满 */
.main-splitter :deep(.el-splitter__panel) {
  overflow: hidden;
  height: 100%;
}
```

## Element Plus Splitter 正确用法

### 组件名称
- 主组件: `el-splitter`
- 面板组件: `el-splitter-panel`

### 属性
- `size`: 面板大小（百分比或像素）
- `min`: 最小大小（像素）
- `max`: 最大大小（像素）

### 示例
```vue
<template>
  <el-splitter>
    <el-splitter-panel size="25%" :min="200">
      左侧面板
    </el-splitter-panel>
    <el-splitter-panel size="50%">
      中间面板
    </el-splitter-panel>
    <el-splitter-panel size="25%" :min="200">
      右侧面板
    </el-splitter-panel>
  </el-splitter>
</template>
```
